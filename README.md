## Run Locally

To run this application locally, use these following commands in the terminal:

* install all dependecies:
    ```
    npm i
    ```
    
* start the server:
    ```
    npm run dev
    ```
* Open this URL in your browser:
    ```
    http://localhost:3000
    ```
